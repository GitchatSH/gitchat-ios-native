# Gitchat iOS Native (Main)

Native iOS port of the [Gitchat extension](https://github.com/GitchatSH/gitchat_extension),
built with SwiftUI. Bundle ID `git.chat`, iOS 16+, syncs with the same backend
(`api-dev.gitstar.ai`, `ws-dev.gitstar.ai`) so your IDE and phone stay in lock-step.

See [walkthrough.md](walkthrough.md) for architecture and build/install steps.

## Quick start

```bash
brew install xcodegen
xcodegen generate
open GitchatIOS.xcodeproj
```
